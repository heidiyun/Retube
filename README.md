
# Re-tube
* 유튜브 댓글 언어별 분류
* 유튜브 댓글 언어 구성 비율, 키워드, 감정 시각화

## 기획의도
기존 유튜브의 한국어 댓글 차별 논란에 따른 언어별 분류 기능에 대한 사용자의 요구를 해소하기 위함.

## 주요기능
* 언어별 댓글 분류
댓글의 언어를 감지하여 언어별로 분류
* 댓글 번역
다양한 외국어 댓글을 한국어로 번역
* 댓글 분석
댓글에 나타나는 키워드, 감정, 언어정보를 분석 후 시각화
